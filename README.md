 Como Rodar o Projeto
1. Importe o arquivo `velocimetro.ino` na IDE do Arduino e grave na ESP32.
2. Coloque a pasta `web` dentro do diretório `htdocs` (XAMPP) ou `www` (WampServer).
3. Crie o banco de dados utilizando o script SQL fornecido.
